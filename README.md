etp
===

Erlang Term Parser


# Usage


	EPTTerm com.metadave.etp.ETP.parse(String s)

	EPTTerm com.metadave.etp.ETP.parse(InputStream is)

An etp term is made up of any of these classes:

- ETPTerm (abstract)	
- ETPAtom
- ETPBinary
- ETPBoolean
- ETPDouble
- ETPFun
- ETPInteger
- ETPList
- ETPPid
- ETPRef
- ETPString
- ETPTerm
- ETPTuple
